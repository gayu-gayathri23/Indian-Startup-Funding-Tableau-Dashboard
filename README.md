# Indian Startup Funding Analysis Dashboard

## 📊 Project Overview
This project presents an interactive Tableau dashboard analyzing startup funding trends across India. The dashboard highlights investment patterns, leading industries, top investors, and city-wise funding distribution.

## 🔧 Tools Used
- Tableau Public
- Data Visualization
- Business Analytics

## 📈 Key Insights
- FoodTech and Consumer Electronics received major funding
- Accel and Zodius Capital are leading investors
- Funding activity peaked around 2024
- Pune, Delhi, and Kolkata emerged as major startup hubs
